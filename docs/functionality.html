Functionality

Base-Learners

Base-Learner Description Arguments
PolynomialBlearner Fit polynomial base-learner. Note that this base-learner takes just the power of the selected feature. Hence, there is no linear part if one selects, for instance, degree = 2. degree: Polynomial degree of feature (default: 1)
intercept: Number of inner knots (default: TRUE)
PSplineBlearner Fit (penalized) spline regression base-learners. degree: Polynomial degree of bases (default: 3)
n.knots: Number of inner knots (default: 20)
penalty: Penalty parameter for smoother curves (default: 2)
differences: Number of penalized differences of the knots (default: 2)
CustomBlearner Define a custom base-learner by using R functions. instantiate.fun: Feature transformation, e.g., create spline bases in the case of a spline learner
train.fun: Function to train on the data gained by instantiate.fun
predict.fun: Function to predict on the object returned by train.fun
param.fun: Function to extract parameters as matrix from the object returned by train.fun
CustomCppBlearner Define a custom base-learner by using C++ functions. instantiate.ptr: External pointer containing the reference to the C++ feature transformation
train.ptr: Function to train on the data obtained by instantiate.ptr which always returns the parameter as matrix predict.ptr: Function to predict by using the parameter obtained by trian.ptr
