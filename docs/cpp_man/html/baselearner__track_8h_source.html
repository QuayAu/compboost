25 // Implementation of tracking all baselearners within the main algorithm.
26 // This is more convenient, multifunctional and (most important) it keeps
27 // the main algorithm cleaner.
28 //
29 // The idea is, that every returned baselearner from the optimizer ìs
30 // registered in a vector of baselearner (the track). Later on we can
31 // predict by using this vector.
