Detailed Description

Quadratic loss for regression tasks.

This loss can be used for regression with \(y \in \mathbb{R}\).

Loss Function:

\[ L(y, f(x)) = \frac{1}{2}\left( y - f(x) \right)^2 \]

Gradient:

\[ \frac{\delta}{\delta f(x)}\ L(y, f(x)) = f(x) - y \]

Initialization:

\[ \hat{f}^{[0]}(x) = \underset{c\in\mathbb{R}}{\mathrm{arg~min}}\ \frac{1}{n}\sum\limits_{i=1}^n L\left(y^{(i)}, c\right) = \bar{y} \]
