Detailed Description

0-1 Loss for binary classification derifed of the binomial distribution

This loss can be used for binary classification. The coding we have chosen here acts on

\[ y \in \{-1, 1\}. \]

Loss Function:

\[ L(y, f(x)) = \log\left\{1 + \exp\left(-2yf(x)\right)\right\} \]

Gradient:

\[ \frac{\delta}{\delta f(x)}\ L(y, f(x)) = - \frac{y}{1 + \exp\left(2yf\right)} \]

Initialization:

\[ \hat{f}^{[0]}(x) = \frac{1}{2}\log\left(\frac{p}{1 - p}\right) \]

with

\[ p = \frac{1}{n}\sum\limits_{i=1}^n\mathbb{1}_{\{y_i > 0\}} \]
