\subsection{Detailed Description}
With this loss it is possible to define custom functions in {\ttfamily C++} 

This one is a special one. It allows to use a custom loss programmed in {\ttfamily C++}. The key is to use external pointer to set the corresponding functions. The big advantage of this is to provide a (not too complicated) method to define custom {\ttfamily C++} losses without recompiling compboost.

{\bfseries Note\+:} This class doesn\textquotesingle{}t have a constructor to initialize a custom offset. Because this is not necessary here since the user can define a custom offset within the {\ttfamily init\+Fun} function.
