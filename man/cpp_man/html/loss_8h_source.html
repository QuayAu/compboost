23 // -------------------
24 //
25 // This file contains the different loss implementations. The structure here
26 // is:
27 // - Parent class 'LossDefinition' which are virtual member functions. This
28 // functions are later overwritten by the child member functions which
29 // contains the concrete implementation of the loss.
30 //
31 // - Child classes have the structure:
32 //
33 // class SpecificLoss: public LossDefinition
34 // {
35 // arma::vec definedLoss { IMPLEMENTATION };
36 // arma::vec definedGradient { IMPLEMENTATION };
37 // double constantInitializer { IMPLEMENTATION };
38 // }
39 //
40 // - There is one special child class, the 'CustomLoss' which allows to
41 // define custom loss functions out of R.
42 //
