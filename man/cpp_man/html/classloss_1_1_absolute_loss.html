Detailed Description

Absolute loss for regression tasks.

Loss Function:

\[ L(y, f(x)) = \left| y - f(x) \right| \]

Gradient:

\[ \frac{\delta}{\delta f(x)}\ L(y, f(x)) = \mathrm{sign}\left( f(x) - y \right) \]

Initialization:

\[ \hat{f}^{[0]}(x) = \underset{c\in\mathbb{R}}{\mathrm{arg~min}}\ \frac{1}{n}\sum\limits_{i=1}^n L\left(y^{(i)}, c\right) = \mathrm{median}(y) \]
